# Requirements

1.1 The program must be able to tell if to given strings are isomorphic or not.

# Acceptance Criteria

1.1 If two strings are isomorphic return true.
1.2 If two strings are not isomorphic return false. 

# Scenarios

1.1 "", "" => true
1.2 "foo", "bar" => false
1.3 "egg", "add" => true
1.4 "paper", "title" => true
1.5 "map", "add" => false
